## üìÑ TROUBLESHOOTING REFERENCE

### Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| Avatar out of sync | Regenerate audio, adjust speed |
| Mispronunciation | Edit script, use hyphens |
| Script too long | Increase speed (1.1x or 1.2x) |
| Background colors wrong | Copy exact hex: #0066ff ‚Üí #7c3aed |
| Logo cut off | Reposition, reduce size, add margin |
| Audio too quiet/loud | Adjust speed, change emphasis |
| Video pixelated | Ensure 1920x1080 resolution |
| Processing slow | Normal, takes 2-3 minutes |
| Download fails | Retry, check internet connection |
| File won't open | Verify MP4 codec (H.264) |
